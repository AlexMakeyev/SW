Swamp Brotherhood

Виробляємо маскувальні засоби для ЗСУ

опис

Swamp Brotherhood — це організація по виготовленню маскувальних засобів для наших захисників. Ми розробляємо та вдосконалюємо маскування. Для цього нам потрібна ваша підтримка.

Для виготовлення маскування нам завжди необхідно різні види тканин, фурнітури, мішків та їх фарбування. Для осінньої пори ми закуповуємо листочки, це надає більший ефект злиття з оточуючим середовищем.

Кожен ваш донат — це безпосередній внесок у допомогу ЗСУ та перемогу України!
